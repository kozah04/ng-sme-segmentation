# Nigerian SME Segmentation

Unsupervised segmentation of Nigerian
small and medium enterprises using K-Means clustering, hierarchical clustering, and PCA.
No target variable - this is a discovery task.

---

## Project Overview

**Goal:** Identify distinct business profiles among formal Nigerian SMEs to support
policy targeting, credit access decisions, and SME support interventions.

**Dataset:** World Bank Nigeria Enterprise Survey 2025. 1,043 formally registered
firms surveyed between February and September 2025 across six geopolitical zones.
Data covers firm scale, financial structure, operations, innovation, business
environment perception, and gender inclusion.

**Methods:** K-Means clustering, agglomerative hierarchical clustering (Ward linkage),
PCA for dimensionality exploration and 2D visualisation, silhouette analysis,
elbow method, Calinski-Harabasz index, Davies-Bouldin index.

---

## Segments Identified

Four segments were identified from 21 features across 1,043 firms.

| Segment | n | Share | Key characteristic |
|---|---|---|---|
| Constrained Micro Firms | 503 | 48.2% | Smallest firms, lowest capability signals, highest obstacle ratings |
| Domestic Mid-Tier | 318 | 30.5% | Domestically oriented, moderate scale, lower constraint intensity |
| Formal Export Leaders | 161 | 15.4% | Internationally connected, quality certified, self-funded |
| Credit-Reliant Traders | 61 | 5.8% | Bank-dependent, import-heavy, high obstacle perception despite credit access |

**Segment naming** is inferred dynamically each run from cluster centroids using
weighted feature scoring and Hungarian assignment. This prevents label drift when
K-Means assigns different cluster IDs across runs.

## Setup

**1. Clone the repo**
```bash
git clone git@github.com:kozah04/ng-sme-segmentation.git
cd ng-sme-segmentation
```

**2. Create the conda environment**
```bash
conda env create -f environment.yml
conda activate ng-sme-segmentation
```

**3. Add the raw data**

Download the Nigeria 2025 Enterprise Survey microdata from the
[World Bank Microdata Library](https://microdata.worldbank.org) (registration required).
Place the `.dta` file at:
```
data/raw/Nigeria-2025-full-data.dta
```

---

## Running the Pipeline

**Full run - generates clean data, all figures, and segmented output**
```bash
python pipeline.py
```

**Skip plot generation for a faster run**
```bash
python pipeline.py --skip-plots
```

**Override defaults**
```bash
python pipeline.py --clusters 5           # use k=5 instead of k=4
python pipeline.py --random-state 0       # different random seed
python pipeline.py --k-min 3 --k-max 12  # wider k sweep
python pipeline.py --raw-data path/to/other.dta  # different input file
```

**Outputs**

| File | Description |
|---|---|
| `data/processed/sme_clean.csv` | Cleaned, encoded, imputed, transformed features |
| `data/processed/sme_segmented.csv` | Clean data with cluster ID and segment name |
| `outputs/reports/segment_profile.csv` | Mean feature values per cluster |
| `outputs/figures/` | All EDA and modelling plots |

## Notebooks

The notebooks document the full exploration journey with every decision explained.
They are the authoritative source of reasoning - the `src/` modules reproduce the
same logic in reusable form.

**`notebooks/eda.ipynb`**
Feature selection from 337 raw columns down to 21, dtype handling for Stata
categoricals, missingness analysis, encoding strategy, imputation, log transforms,
variance checks, and correlation analysis.

**`notebooks/modelling.ipynb`**
StandardScaler, PCA variance analysis, elbow and silhouette evaluation across
K=2 to K=9, K-Means at K=4, silhouette plot, 2D PCA projection, Ward hierarchical
clustering, method comparison, cluster profiling, radar charts, sector and region
enrichment from survey metadata.

---

## Key Decisions

| Decision | Reason |
|---|---|
| 21 features from 337 columns | Selected across 6 business dimensions - scale, finance, operations, innovation, environment, gender |
| Dropped `f1` capacity utilization | 58.7% missing - above reliable imputation threshold |
| Dropped `l5` women employees | 41.3% missing - `b4` women owners covers gender angle cleanly |
| Log-transformed sales and employees | Heavy right skew - trillion-scale sales would dominate distance |
| Replaced `d2` and `n3` with `sales_growth` | Correlation of 0.94 - near-duplicate signal |
| Full features for clustering | PCA PC1 explains only 14.4% - no dominant axis |
| PCA for 2D visualisation only | 2 components explain 23.3% - rough picture only |
| K=4 over K=2 | K=2 maximises silhouette but yields only big vs small - not actionable |
| K-Means over hierarchical | Better on all three evaluation metrics |
| Dynamic segment naming | Prevents label drift when cluster IDs swap between runs |

## Limitations

- **Stratified sample:** The 1,043 firms are not a random sample. The World Bank
  used stratified sampling across region, size, and sector. Results represent
  formal registered firms, not the informal sector which dominates the Nigerian economy.
- **South West over-representation:** 344 of 1,043 firms (33%) are from the South West,
  reflecting its higher density of formal businesses. Regional distributions within
  segments should be interpreted with this in mind.
- **Moderate cluster separation:** Silhouette score of 0.083 reflects that Nigerian
  SMEs vary across many independent dimensions simultaneously rather than grouping
  into tight clusters. This is consistent with multi-dimensional survey data and
  documented in comparable studies on Nigerian MSMEs.
- **Survey year:** Data was collected in 2025. Segment profiles reflect the current
  business environment including post-COVID recovery conditions and ongoing
  infrastructure constraints.
